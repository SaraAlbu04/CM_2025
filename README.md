# GhostMappers

GhostMappers is an Android application that allows users to report, track, and interact with ghost sightings in real-world locations. Built with Jetpack Compose and Firebase, this spooky social app gamifies paranormal encounters by letting users document and explore supernatural activities around them.

## Features

### Core Functionality

- **Ghost Reporting**: Capture photos of ghost sightings with location and environmental data
- **Interactive Map**: Real-time map showing ghost locations with navigation support
- **Ghost Classification**: Multiple ghost types with different visual appearances
- **Profile Management**: User profiles with statistics and achievements
- **Real-time Notifications**: Background service alerts users to nearby ghost activity

### Map Features

- **Mapbox Integration**: High-quality maps with custom ghost markers
- **Navigation**: Directions to ghost locations
- **Ghost States**: Visual indicators for different ghost statuses
- **Kill Mode**: Interactive ghost "vacuuming" feature
- **Real-time Updates**: Live ghost data synchronization

### Reporting System

- **CameraX Integration**: Photo capture for ghost evidence
- **Location Tracking**: GPS coordinates for precise ghost positioning
- **Environmental Data**: Temperature recording at time of sighting
- **Audio Recording**: Optional audio evidence capture
- **Ghost Type Selection**: Categorize each ghost

### User Experience

- **Firebase Authentication**: Secure user login and registration
- **Profile Statistics**: Track reports, kills, and ghost types encountered
- **Achievements System**: Unlock rewards based on ghost hunting activities
- **Customizable Settings**: Notification preferences
- **Profile Pictures**: Custom or preset profile image options
